# 瓦片切割工具

将需要切割的图片拖拽到 slice.exe 上即可.

### 配置文件
在图片旁边新建一个 `config.ini` 文件, 在里面写入
```ini
[settings]
tileSize = 256  ; 瓦片大小
Quality = 90    ; 质量 0-100
```

### 编译

使用 [xmake](https://xmake.io/#/zh-cn/) 进行编译


### 瓦片
